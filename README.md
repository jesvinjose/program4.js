# program4.js
A program to count how many numbers in a list are divisible by numbers from 1 to 9

It is a program that takes a list of integers (entered by the user) and checks how many of them are divisible by each number from 1 to 9.

The result is shown as a dictionary where:

The key is a number from 1 to 9,

The value is how many numbers in the list are divisible by that key.

Steps to run the code:

1. Clone the repository:

2. Make sure Node.js is installed (v17 or higher): node -v

3. Open the terminal in the folder (repo) where program4.js file is saved.

4. Run the program: node program4.js

5. Enter the list of numbers when prompted (comma-separated).

6. View the output in the terminal.
